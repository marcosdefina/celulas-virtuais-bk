:: 1) Introdução :: [Menu]

"Células Virtuais" é um software educacional que objetiva o complemento do ensino de biologia celular a alunos do ensino fundamental. Em sua primeira versão procurou-se exigir o mínimo de leitura diante do computador, o que pode se tornar cansativo e desestimulante. Dessa forma grande parte do conteúdo escrito foi substituído por narrações.

As imagens e animações geradas são esquemáticas, visando uma abordagem mais didática.

Também se procurou incentivar a utilização do programa pelo aluno ao criar um ambiente semelhante a um jogo durante a resolução de exercícios, onde a cada exercício resolvido corretamente o aluno vai acumulando pontos. Ao entrar no programa solicita o nome do usuário, ao sair o nome fornecido é utilizado para o tratamento pessoal do usuário. Ainda ao sair do programa o usúario fica sabendo a pontuação atingida.

O programa é todo baseado em ligações, possibilitando ao aluno desenvolver sua utilização de forma não linear (pré-estabelecida). Essas ligações podem conduzir a textos, imagens, sons e vídeos que possuem relação com o item anterior. Geralmente os componentes que possuem ligações são facilmente identificadas por se apresentarem de alguma forma destacadas (através da cor, sensibilidade ao passar do mouse, forma de botões, figuras sugestivas, etc).



:: 2) Considerações a respeito desta versão :: [Menu]

Dentre os diversos tipos celulares que podem ser encontrados nos organismos vivos a versão 1.5 de "Células Virtuais" limita-se apenas as células eucarióticas animais, eucarióticas vegetais e procarióticas. Outros tipos celulares mais específicos poderão ser incluídos em versões futuras.



:: 3) Requerimentos mínimos :: [Menu]

"Células Virtuais" necessita de um computador com CD-ROM, Kit Multimídia e um navegador compatível com o Microsoft Internet Explorer ® ou com o Netscape Communicator ® (como o Mozilla Firefox ®), com suporte a Javascript Ativado e plugin QuickTime ou RealPlayer (RealOne).



:: 4) A navegação :: [Menu]

Navegação é o termo que se usa para designar a migração entre documentos ou páginas através ligações (hipertexto ou hipermídia).
Através da navegação é possível para o usuário seguir uma seqüência, um roteiro, que será montada de acordo com seu interesse e necessidades. Nesse programa em específico a navegação acontece ao clicar sobre uma imagem (ou porção delimitada desta), palavras-chave, animações ou botões. Desses os mais freqüentes são os botões.
